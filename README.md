# Botanic Restaurant WordPress Theme

A custom WordPress child theme for Botanic Restaurant, built on the Neve parent theme.

## Requirements

- WordPress 5.0+
- Node.js 14+
- npm 6+
- Neve parent theme

## Installation

1. Install the Neve parent theme
2. Clone this repository into wp-content/themes/
3. Install dependencies:

```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Scripts

- `npm run dev`: Start development environment
- `npm run build`: Build for production
- `npm run lint`: Run linters
- `npm run lint:fix`: Fix linting errors
